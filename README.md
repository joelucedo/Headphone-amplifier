# Headphone-amplifier

My build of the butte headphone amplifier from Pete Millett : http://www.pmillett.com/butte.htm

## Specs and features

* Power supply : 12VAC 1A
* Max output current : 200mA par canal
* Input 1 : RCA
* Input 2 : TRS
* input selection switch
* Headphone output
* Volume control
* ON/OFF switch and power led
